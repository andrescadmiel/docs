## Headers

Headers are used to organize your content and create a hierarchy.

```markdown
# H1 Header
## H2 Header
### H3 Header
```

## Text Formatting

Make text **bold**, _italic_, or ~~strikethrough~~.

```markdown
**Bold text**
*Italic text*
~~Strikethrough text~~
```

## Lists

Create ordered and unordered lists:

### Unordered List
- Item 1
- Item 2
  - Nested item
  - Another nested item

### Ordered List
1. First item
2. Second item
3. Third item

## Code

### Inline Code
Use `backticks` for inline code.

### Code Blocks
```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
}
```

### Code Groups
<CodeGroup>

```javascript JavaScript
const greeting = "Hello World";
console.log(greeting);
```

```python Python
greeting = "Hello World"
print(greeting)
```

</CodeGroup>
